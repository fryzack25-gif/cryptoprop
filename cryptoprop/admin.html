How it works

        
    
          
  1. Click Enter Beta Mode — your session is now impersonating the beta account
    2. 
          
  2. Open the Trading page or Dashboard in a new tab
    3. 
          
  3. All trades, data, and API calls will use the beta account ($100K funded)
    4. 
          
  4. When done testing, come back here and click Exit Beta Mode
    5. 
          
  5. Use Reset Beta Account any time to wipe positions and start fresh
    6. 
        

        
⚠ Beta mode is session-scoped — it resets if you log out or your session expires.
